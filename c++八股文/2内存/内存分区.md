+-----------------------+ ← 高地址
|     栈区 (Stack)       | ↓ 向低地址增长
+-----------------------+
|     堆区 (Heap)        | ↑ 向高地址增长
+-----------------------+
| 未初始化数据段 (BSS)   |
+-----------------------+
| 初始化数据段 (Data)    |
+-----------------------+
| 代码段 (Text/Code)     |
+-----------------------+ ← 低地址


区段名	    内容	                                          示例变量
.text	      可执行代码（函数体）	                            int add(int a, int b)
.rodata	    只读数据（字符串字面量、const常量）	              const int x = 5;、"abc"
.data	      初始化的全局/静态变量	                            int a = 10;
.bss	      未初始化的全局/静态变量	                          static int b;
heap	      动态分配的内存区域	                              malloc/new
stack	      函数局部变量、参数、返回地址等	                    int local = 0;

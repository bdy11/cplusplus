说说内联函数和宏函数的区别

1. 内联函数在编译时有类型检查，问宏函数没有，只是纯粹文本替换
2. 作用机制上，内联函数在编译阶段函数体展开，由编译器控制是否内联，宏函数在预处理阶段文本替换
3. 内联函数可调试，宏函数不可调试
4. 内联函数语法安全


宏函数示例(潜藏问题)
#define MAX(a, b) ((a) > (b) ? (a) : (b))
int x = MAX(3, 5);  // 正确

int y = MAX(x++, 5);
等价为 int y = x++ > 5 ? x++ : 5;  // ❌ 会导致 x++ 被执行两次！


内联函数示例(安全可调试)
inline int max(int a, int b) {
    return (a > b) ? a : b;
}
int x = max(3, 5);
int y = max(x++, 5);  // ✅ 正常只执行一次 x++
